.. _installation:

============
Installation
============

Appollo is a CLI utility developed in Python. It is easily installed with pip.

.. code-block::

    pip install appollo


Once installed we can easily run the CLI in the terminal :code:`$ appollo`

.. note::

    PIP is the python package manager. If pip or python is not installed on your computer, check : https://packaging.python.org/en/latest/tutorials/installing-packages/.

If it is your first time using Appollo, create an Appollo account by running :code:`$ appollo signup`.
